title: Adagio
description: Build and run bioinformatics pipelines without writing code
---

Adagio lets you build and run bioinformatics data processing pipelines through a visual interface — no programming required.

Connect analysis steps together on a canvas, configure your inputs, and run your pipeline directly from the browser or from the command line.

## Where to start

- **New to Adagio?** Start with the [Introduction](/getting-started/introduction/) to understand what Adagio does and how it works.
- **Ready to build?** Jump to the [Quick Start](/getting-started/quick-start/) to build and run your first pipeline.
- **Want to run pipelines from the terminal?** See [Running with the CLI](/running/cli/).
- **Building a custom plugin or pipeline?** See the [Contributing](/contributing/building-a-plugin/) section.
